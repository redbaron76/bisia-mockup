# Bisia Mockups

5 proposte di landing page con design moderno, tema scuro e palette nero/grigio scuro con accenti bianchi e giallo `#ffcc00`.

## Landing Pages

| # | File | Stile | Typography |
|---|---|---|---|
| 1 | `landing-1-bold.html` | Bold & Angular | Inter + Space Grotesk |
| 2 | `landing-2-elegant.html` | Elegant & Clean | Playfair Display + DM Sans |
| 3 | `landing-3-tech.html` | Tech & Grid | JetBrains Mono + Inter |
| 4 | `landing-4-warm.html` | Warm & Human | Outfit + Plus Jakarta Sans |
| 5 | `landing-5-luxury.html` | Luxury & Dark | Cormorant Garamond + Montserrat |

## Stack

- **HTML5** + **CSS3** — zero dipendenze esterne (tranne Google Fonts)
- **Responsive** — mobile-first con breakpoint a 768px e 900px
- **Dark theme** — `#0a0a0a` / `#050505` come base, `#ffcc00` come accent

## Come usare

1. Apri qualsiasi file `.html` nel browser
2. Nessuna build necessaria — tutto è inline (HTML + CSS)

```bash
# Con live server (VS Code)
open landing-1-bold.html

# Oppure con Python
python3 -m http.server 8080
```

## Palette

- **Background:** `#0a0a0a` / `#050505` / `#000`
- **Card:** `#141414` / `#111` / `#0a0a0a`
- **Borders:** `#1e1e1e` / `#1a1a1a`
- **Text primary:** `#ffffff`
- **Text secondary:** `#888888` / `#666666`
- **Accent:** `#ffcc00`
- **Accent hover:** `#ffe033`
